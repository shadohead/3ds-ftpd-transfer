3DS ftpd Transfer Guide

A practical guide for installing the tool, opening ftpd, sending files to your 3DS, loading NDS cheat databases, and setting up DS forwarder workflows.

Legal use only.

Use this tool for homebrew, saves, themes, patches, cheat databases, and game backups you are legally allowed to copy. This project does not include ROM download, archive scraping, or copyrighted-content search features.

Install Without Python or Homebrew

macOS / Linux

Installs one executable to ~/.local/bin.

curl -fsSL https://raw.githubusercontent.com/shadohead/3ds-ftpd-transfer/main/scripts/install.sh | sh

Uninstall:

curl -fsSL https://raw.githubusercontent.com/shadohead/3ds-ftpd-transfer/main/scripts/uninstall.sh | sh

Windows PowerShell

Installs one executable to %LOCALAPPDATA%\Programs\3ds-ftpd-transfer.

irm https://raw.githubusercontent.com/shadohead/3ds-ftpd-transfer/main/scripts/install.ps1 | iex

Uninstall:

irm https://raw.githubusercontent.com/shadohead/3ds-ftpd-transfer/main/scripts/uninstall.ps1 | iex

Direct downloads are also available from the latest GitHub Release.

Send a File to Your 3DS

  1. Install ftpd on your 3DS if it is not already installed. Universal-Updater is the easiest way after finalizing CFW setup.
  2. Open ftpd on the 3DS and leave it running.
  3. Note the IP address and port shown on the 3DS screen. The port is usually 5000.
  4. Run 3ds-ftpd-transfer on your computer. It opens a local browser page.
  5. Enter the 3DS IP address and port.
  6. Choose a destination folder, such as /3ds/inbox, /roms/nds, /cias, or /luma/titles.
  7. Choose a local file or paste a direct legal/homebrew URL.
  8. Click Send to 3DS.

Load NDS Cheats for TWiLight Menu++

The app has a dedicated NDS cheats for TWiLight Menu++ panel. It installs usrcheat.dat to the path used by TWiLight Menu++ and nds-bootstrap:

/_nds/TWiLightMenu/extras/usrcheat.dat
  1. Download a DeadSkullzJr-style usrcheat.dat, or an archive containing it. The source page is DeadSkullzJr NDS Cheat Databases on GameBrew.
  2. Open ftpd on the 3DS.
  3. Run 3ds-ftpd-transfer on your computer.
  4. Enter the IP and port from ftpd.
  5. In the NDS cheats panel, choose usrcheat.dat, a .zip, or a .7z containing usrcheat.dat.
  6. Click Install Cheats for TWiLight Menu++.

Enable cheats in TWiLight Menu++

  1. Open TWiLight Menu++.
  2. Highlight the DS game.
  3. Press Y for per-game settings.
  4. Press X to open the cheat menu.
  5. Select cheats, save, and launch the game.

NDS Forwarders

NDS forwarders are HOME Menu icons that launch DS games through nds-bootstrap. The recommended path is to install NDS Forwarder through Universal-Updater and keep nds-bootstrap current.

  1. Install or update TWiLight Menu++ and nds-bootstrap.
  2. Install NDS Forwarder from Universal-Updater.
  3. Put your legally dumped .nds files somewhere stable, such as /roms/nds.
  4. Generate/install forwarders for those files using NDS Forwarder.

Using cheats with forwarders

Modern forwarder setups can use the same TWiLight Menu++ cheat database path:

/_nds/TWiLightMenu/extras/usrcheat.dat
  1. Install the cheat database using this tool.
  2. Launch the forwarder while holding Y.
  3. Keep holding Y until per-game settings appear.
  4. Press X to open the cheats menu.

Some older forwarder setups also check /_nds/ntr-forwarder/usrcheat.dat. If your forwarder does not see cheats, update NDS Forwarder and nds-bootstrap first.

SD Card Prep and CFW Finalizing

This app is mainly for daily file transfer after your 3DS is already set up. For initial custom firmware installation, follow the current 3DS Hacks Guide. Entry points and files change over time, so do not rely on old screenshots or outdated videos.

Common SD card format

  • Partition map: MBR is the safest default for 3DS compatibility.
  • Filesystem: FAT32.
  • Cluster size: 32 KB is a good default for larger cards.
  • Avoid exFAT for 3DS SD cards.

After installing CFW

  1. Finish the official finalizing setup steps.
  2. Make and save your NAND backup.
  3. Install core apps such as FBI, Universal-Updater, Checkpoint, ftpd, Anemone3DS, and GodMode9.
  4. Copy the /gm9/backups/ folder from the SD card to a safe place on your computer or cloud storage.

Could this tool format and prepare an SD card?

A local desktop helper could do this, but a hosted website cannot safely erase or repartition drives. Formatting is destructive, OS-specific, and should require an explicit local app flow with confirmation. This project may add a local-only SD prep wizard later, but the web docs will not attempt to format drives.

Why This Cannot All Run From a Hosted Website

A normal hosted website is sandboxed by the browser. That is good for privacy, but it limits hardware and local-network access.

TaskHosted Website?Why
Show instructions and download linksYesStatic web pages are ideal for this.
Let the user choose a file manuallyPartlyBrowsers can read files only after the user picks them.
Write arbitrary SD card foldersNoBrowsers cannot browse and write local drives freely.
Format an SD cardNoBrowsers cannot repartition or erase drives.
Upload files over FTP to ftpdNoBrowsers do not expose raw FTP client APIs to JavaScript.
Run local helper logicYes, with this appThe installed app runs locally and opens a browser UI.

That is why this project uses a local helper app: the UI still feels like a website, but the executable can download files, extract archives, and connect to the 3DS over FTP.

Troubleshooting

The app cannot connect to ftpd

  • Make sure the 3DS and computer are on the same Wi-Fi network.
  • Make sure ftpd is open and the 3DS is awake.
  • Check the IP and port exactly as shown on the 3DS.
  • Temporarily allow the app through your firewall if prompted.

Cheats do not appear

  • Confirm the file is named exactly usrcheat.dat.
  • Confirm it is at /_nds/TWiLightMenu/extras/usrcheat.dat.
  • Update TWiLight Menu++, nds-bootstrap, and your forwarder pack.
  • For forwarders, hold Y while launching, then press X in per-game settings.

macOS says the app cannot be verified

The macOS binaries are not Apple-notarized yet. Use the install script, or remove quarantine from a direct download:

xattr -dr com.apple.quarantine ./3ds-ftpd-transfer
chmod +x ./3ds-ftpd-transfer
./3ds-ftpd-transfer
